# 📊 PhonePe Payment Insights Dashboard

## Overview
This project presents an interactive Power BI dashboard designed to analyze transaction patterns, customer behavior, service adoption, and payment performance within the PhonePe ecosystem.

The objective of this project is to transform raw transactional data into actionable business insights through data visualization, KPI monitoring, and storytelling.

---

## Why PhonePe?

PhonePe is one of India's leading digital payment platforms with millions of active users and a diverse range of financial services.

I selected PhonePe as the focus of this project because it provides an excellent business use case for exploring:

• Digital payment trends  
• Customer engagement patterns  
• Service utilization analysis  
• Operational performance metrics  
• User segmentation and behavioral insights  

This dashboard demonstrates how analytics can support strategic decision-making in fintech businesses.

## Dashboard Features

### Executive KPI Section
Provides a high-level overview of platform performance.

### Transaction Trend Analysis
Tracks transaction activity across different months to identify growth patterns and seasonality.

### Age Segment Contribution
Analyzes customer distribution among different demographic groups.

### Service-wise Transaction Analysis
Evaluates performance across services including:

- Money Transfer
- Recharge
- Insurance
- Loan Payments
- Bill Payments

### Top User Analysis
Identifies high-value users contributing significantly to transaction volume.

### Weekday vs Weekend Usage
Compares user activity behavior across weekdays and weekends.

### Interactive Filters
Implemented dynamic slicers for:

- Month Selection
- Payment Status

### Custom Tooltip Pages
Designed and integrated **two interactive tooltip pages** to provide additional contextual insights without increasing dashboard complexity.

These tooltips enhance user experience by enabling detailed exploration through hover interactions.
